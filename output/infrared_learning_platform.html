Interactive Code Examples

Run these examples in your Python environment to start analyzing thermal data

quick_start.py 
# Install dependencies first:
# pip install requests numpy pandas geopandas rasterio matplotlib

from infrared_config import setup_infrared
from infrared_client import InfraredClient
from thermal_analysis import ThermalDataProcessor

# One-line setup (interactive configuration)
config = setup_infrared()

# Create API client and processor
client = InfraredClient(config)
processor = ThermalDataProcessor(client)

# List available simulations
simulations = client.list_simulations()
for sim in simulations:
    print(f"{sim['name']}: {sim['status']}")

# Load first completed simulation
sim_id = simulations[0]['id']
results = processor.load_simulation_results(sim_id)

# Calculate key metrics
stats = processor.calculate_basic_statistics(results)
print(f"Mean UTCI: {stats['mean']:.1f}°C")

stress = processor.calculate_stress_percentages(results)
print(f"Heat stress: {stress['total_heat_stress']:.1f}%")

# You're ready to analyze!
print("✓ Setup complete. Now explore the other modules.")
basic_analysis.py 
import pandas as pd
import matplotlib.pyplot as plt

# Assuming you've already loaded results (see Quick Start)
# Convert to DataFrame for easy analysis
df = processor.results_to_dataframe(results)

# Basic Statistics
print("=" * 60)
print("THERMAL COMFORT ANALYSIS")
print("=" * 60)

# UTCI distribution
print(f"\nUTCI Statistics:")
print(f"  Mean:    {df['utci'].mean():.1f}°C")
print(f"  Median:  {df['utci'].median():.1f}°C")
print(f"  Std Dev: {df['utci'].std():.1f}°C")
print(f"  Range:   {df['utci'].min():.1f}°C to {df['utci'].max():.1f}°C")

# Thermal stress distribution
print(f"\nThermal Stress Distribution:")
comfortable = (df['is_comfortable'].sum() / len(df)) * 100
print(f"  Comfortable (9-26°C): {comfortable:.1f}%")

heat_stress = ((df['utci'] > 26).sum() / len(df)) * 100
print(f"  Heat stress (>26°C):  {heat_stress:.1f}%")

# Visualize distribution
plt.figure(figsize=(10, 6))
df['utci'].hist(bins=50, color='#2c5f2d', alpha=0.7)
plt.axvline(26, color='red', linestyle='--', label='Heat stress threshold')
plt.xlabel('UTCI (°C)')
plt.ylabel('Frequency')
plt.title('UTCI Distribution Across Site')
plt.legend()
plt.show()

print(f"   Spatial Diversity:    {assembly['spatial_diversity']:.2f} (Shannon)")
print(f"   Edge Complexity:      {assembly['edge_complexity']:.2f}")

# Design recommendations based on complexity
if assembly['index'] < 30:
    print(f"\n💡 Design Priority: INCREASE microclimate diversity")
    print("   • Vary vegetation heights (canopy, understory, ground)")
    print("   • Create thermal gradients (sun/shade, wet/dry)")
    print("   • Integrate water features")
elif assembly['index'] < 60:
    print(f"\n💡 Design Priority: ENHANCE existing diversity")
    print("   • Add complementary vegetation layers")
    print("   • Create temporal variation (deciduous + evergreen)")
else:
    print(f"\n✓ High complexity - Good ecological foundation")
    print("   • Protect existing microclimates")
    print("   • Monitor ecological indicators")
qgis_export.py 
from infrared_workflows import QGISExporter
from pathlib import Path

# Initialize exporter
exporter = QGISExporter(processor)

# Create output directory
output_dir = Path('data/outputs')
output_dir.mkdir(parents=True, exist_ok=True)

# Export GeoTIFF (raster heatmap for thermal visualization)
print("Exporting GeoTIFF raster...")
tif_path = exporter.export_geotiff(
    simulation_id=sim_id,
    output_path=output_dir / 'marseille_utci.tif',
    variable='utci',
    resolution=5.0  # 5-meter grid cells
)
print(f"✓ Raster exported: {tif_path}")

# Export GeoJSON (vector points with all attributes)
print(f"\nExporting GeoJSON vector...")
geojson_path = exporter.export_geojson(
    simulation_id=sim_id,
    output_path=output_dir / 'marseille_thermal_points.geojson',
    include_categories=True  # Include thermal stress categories
)
print(f"✓ Vector exported: {geojson_path}")

# QGIS import instructions
print(f"\n" + "="*60)
print("QGIS IMPORT INSTRUCTIONS")
print("="*60)
print(f"1. Open QGIS and create new project")
print(f"2. Add Raster Layer: {tif_path.name}")
print(f"   → Properties → Symbology → Singleband Pseudocolor")
print(f"   → Color ramp: Spectral (reversed)")
print(f"3. Add Vector Layer: {geojson_path.name}")
print(f"   → Symbolize by: 'thermal_category' attribute")
print(f"4. Overlay with building footprints for spatial analysis")

UTCI Fundamentals & Thermal Comfort

Learning Outcomes

  • Categorize UTCI values into thermal stress levels
  • Calculate heat stress percentages for design assessment

MODULE 7 · NEW

infrared-sdk 0.4.9 — Submit & Run

25 minutes Core update

The official infrared-sdk replaces the hand-rolled REST client of Modules 1–6. The mental model has changed: instead of fetching results from a pre-existing simulation by simulation_id, you now submit a geometry and an analysis request, and the SDK tiles the area, runs the jobs, polls them, and returns a merged NumPy grid. The API key is read from the INFRARED_API_KEY environment variable (a local .env is auto-loaded).

Learning Outcomes

  • Construct an InfraredClient() and a typed *ModelRequest
  • Run an analysis with run_area_and_wait(request, polygon)
  • Inject real buildings, weather and vegetation layers
  • Read results from AreaResult.merged_grid (no manual JSON parsing)
import infrared_sdk as ir
from infrared_sdk.models import TimePeriod

client = ir.InfraredClient()                 # reads INFRARED_API_KEY

polygon = {"type": "Polygon", "coordinates": [[...]]}   # your GeoJSON site
weather = client.weather.get_weather_file_from_location(lat=43.30, lon=5.37)
buildings = client.buildings.get_area(polygon).buildings   # context geometry

req = ir.UtciModelRequest(
    latitude=43.30, longitude=5.37,
    analysis_type="thermal-comfort-index",
    time_period=TimePeriod(start_month=8, start_day=15, start_hour=9,
                           end_month=8, end_day=15, end_hour=18),
)
area = client.run_area_and_wait(req, polygon, buildings=buildings)
grid = area.merged_grid          # 2-D NumPy array, NaN outside the polygon
Gotcha: thermal-comfort-index rejects a request with "At least one of [geometries, vegetation] must contain geometry data" if you run it on a bare tile — always inject buildings (and/or vegetation) and supply the weather arrays.

MODULE 8 · CAPSTONE

Regenerative Microclimate Scorecard

40 minutes Multi-physics

The capstone runs all three focus analyses — UTCI, wind and solar — over one site and compares a baseline (buildings only) against a regenerative intervention (real tree canopy injected via the SDK). Each result grid is scored with the regenerative metrics from Modules 3–4 (heat-stress %, Thermal Assembly Complexity Index), now computed grid-native.

Learning Outcomes

  • Orchestrate multi-physics analyses in one workflow
  • Model an intervention by injecting client.vegetation.get_area(polygon).features
  • Quantify the regenerative delta (heat-stress, assembly complexity)
  • Render a one-page scorecard figure for design review

Live result (Marseille, Vieux-Port): buildings 968, trees 579 — adding canopy moved heat-stress area 92.1% → 91.3% and assembly complexity +6.3%.

QGIS Integration Workflow

Step-by-step guide to importing and analyzing thermal data in QGIS

1

Export Georeferenced Data

Use the Python toolkit to export thermal data as GeoTIFF rasters and GeoJSON vectors. The toolkit automatically handles coordinate reference systems (Lambert 93 for Marseille).

exporter.export_geotiff(sim_id, 'marseille_utci.tif', resolution=5.0)
2

Import Raster Layer

Open QGIS and import the GeoTIFF file. The coordinate system is embedded in the file metadata, so QGIS will automatically recognize the projection.

Layer → Add Layer → Add Raster Layer → Select .tif file
3

Symbolize Thermal Data

Right-click the layer and select Properties → Symbology. Choose "Singleband Pseudocolor" and select the "Spectral" color ramp (reversed). Blue = cold, green = comfortable, red = hot.

Properties → Symbology → Singleband Pseudocolor → Spectral (Reversed)
4

Spatial Overlay Analysis

Import building footprints and perform spatial queries. Example: "Which buildings have >50% of their area in heat stress zones (UTCI >26°C)?" Use Vector → Geoprocessing Tools → Intersection.

Vector → Geoprocessing Tools → Intersection → buildings × thermal_zones
5

Export Maps for Dossier

Use Print Layout to create publication-quality maps. Add scale bar, north arrow, legend, and attribution. Export as high-resolution PDF or PNG for Technical Dossier inclusion.

Project → New Print Layout → Add Map → Export as PDF (300 DPI)
