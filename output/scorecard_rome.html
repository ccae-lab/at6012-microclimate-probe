What this demonstrates. The methodological win is data provenance: the model now runs on an authoritative municipal tree inventory with real species, replacing the SDK's empty OSM attributes. The thermal delta is small and slightly negative — the same dense-urban pattern seen in Marseille (canopy masks grid cells and can trap canyon heat). With species now attached, the next step is species-aware canopy: mapping each species to crown size and leaf cycle (a stone pine ≠ an oleander) so the shade model reflects the real tree, not a generic point.

Method

One run_area_and_wait() per analysis over the ~800 m polygon; SDK buildings + nearest-EPW weather for all runs; intervention vegetation from city_data.rome_street_trees() formatted into the SDK injection schema. Reproduce: python scorecard.py --run --site rome --veg-source city.
